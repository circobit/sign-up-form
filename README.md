## Sign-up Form

This projects is part of the [Intermediate HTML and CSS course](https://www.theodinproject.com/paths/full-stack-javascript/courses/intermediate-html-and-css) from [The Odin Project](https://www.theodinproject.com).

The objective is to create a simple sign-up page that uses the input validation attributes to give visual feedback to the users.

As an extra personal point not requested by TOP, I tried to play a bit with responsiveness making use of `@media` queries.
